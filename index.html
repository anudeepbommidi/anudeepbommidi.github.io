June 2017

Project link

An interactive & responsive Full Stack Web Application for football fans. User can view the information of his favorite player and can login and create his own team.

Features:
  1. Login/Signup functionality including Facebook/Google/Twitter OAuth
  2. Search a player directly from search bar
  3. Attractive Bootstrap UI Components

  5. Ability for a logged in user to make his own favorite team of players.
  6. Advanced Mongoose - MongoDB query methods for efficiency.

CarspecsIn - MEAN Stack Project

August 2016
Project Link

Founded, developed and deployed a Full Stack Web Application that allows a user to search, view, compare & filter cars. It is live at http://www.carspecs.in and technologies made use of are Bootstrap, Angular, Express.js, MongoDB and Node.js.

Features:
  • Mobile First Design made possible by the Bootstrap library.
  • Single Page Application paradigm made possible by the AngularJS framework.
  • Event driven server-side programming using Node.js & Express.js for excellent performance.
  • JSON formatted data stored using MongoDB for easy storage and retrieval.
  • Hosted currently in Amazon Web Services.
